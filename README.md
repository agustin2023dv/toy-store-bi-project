# toy-store-bi-project
End-to-end Business Intelligence project on a toy store e-commerce dataset. Data exploration, cleaning, and advanced analytics with Python (pandas, numpy, seaborn, scikit-learn). Interactive dashboard built with Power BI. Includes KPIs, trend forecasting, customer segmentation, and actionable insights.

## Setup 

1. Create and activate a virtual environment (see commands above).
2. Install dependencies: `pip install -r requirements.txt`
3. Download the dataset from Maven Analytics and place the CSV files in `data/raw/`
4. Run the Jupyter notebooks in order.
